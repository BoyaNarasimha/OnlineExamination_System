# OnlineExamination_System

Online Examination System is a web-based application developed to conduct and manage online examinations efficiently. The system allows users to register, attend exams, manage questions, and view results in a simple and organized way.
The project is built using Java, JSP, Servlets, JDBC, and MySQL, providing a user-friendly interface for handling examination activities with secure authentication and CRUD operations.

---------------------------------------------------------------------------
# Features

User Registration and Login
Secure Authentication
Online Exam Management
Question Management
Result Generation

Configure MySQL database
Update database credentials
Run the application on Apache Tomcat Server
Open browser and run the application
